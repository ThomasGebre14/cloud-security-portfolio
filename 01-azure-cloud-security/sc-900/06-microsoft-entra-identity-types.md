# 🌐 Microsoft Entra Identity Types

## Common Identity Types Supported by Microsoft Entra
- Workforce identities
- Guest / external identities
- Application identities
- Device identities

## Hybrid Identity
Microsoft Entra can work with on-premises Active Directory to provide a hybrid identity model.

Benefits:
- consistent user identity across cloud and on-prem
- centralized access control
- smoother user experience
- support for legacy and modern environments

## External Collaboration
Microsoft Entra supports secure collaboration with users outside the organization, such as partners, guests, and customers.
