# 📋 Service Trust Portal, Priva, and Purview

## Microsoft Service Trust Portal
The Service Trust Portal provides:
- audit reports
- whitepapers
- compliance documentation
- industry and regional resources
- resources for an organization’s compliance program

## Microsoft Privacy Principles
Microsoft positions privacy as a core trust area and provides tools and documentation to help organizations meet privacy obligations.

## Microsoft Priva
Microsoft Priva helps organizations manage privacy operations across their digital estate.

### Key solutions:
- Subject Rights Requests
- Privacy Risk Management

### Privacy Risk Management examples:
- limit data overexposure
- detect risky data transfers
- minimize stored data

## Microsoft Purview
Microsoft Purview is a broad platform for data security, data governance, and compliance.

### Key data security capabilities:
- Information Protection
- Data Loss Prevention (DLP)
- Insider Risk Management

### Data Classification in Purview
Purview supports:
- sensitive information types
- exact data match (EDM)
- trainable classifiers

### Insider Risk Management
Helps detect, investigate, and respond to internal risk such as:
- data leaks
- IP theft
- fraud
- policy violations
