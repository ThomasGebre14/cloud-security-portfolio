# 👤 Identity Concepts

## Authentication vs Authorization
### Authentication (AuthN)
Proves who you are.
Examples:
- username and password
- MFA
- passkeys

### Authorization (AuthZ)
Determines what you are allowed to access after authentication.

## MFA and Passwordless
### Multifactor Authentication
Combines multiple factors:
- something you know
- something you have
- something you are

### Passwordless
Reduces dependence on passwords and can improve phishing resistance.

## Identity as the New Security Perimeter
Organizations no longer rely only on the network boundary.
Identity now protects access to:
- cloud apps
- on-prem apps
- partner access
- BYOD devices
- remote users
- IoT devices

## Four Pillars of Identity Infrastructure
- Administration
- Authentication
- Authorization
- Auditing
