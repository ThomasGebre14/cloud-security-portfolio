# 🔑 Microsoft Entra Fundamentals

## What is Microsoft Entra ID?
Microsoft Entra ID is Microsoft’s cloud-based identity and access management service.

It helps organizations manage access to:
- internal apps
- Microsoft cloud services
- SaaS applications
- hybrid resources

## Core Identity Capabilities
- Authentication
- Authorization
- Single sign-on (SSO)
- Multifactor authentication (MFA)
- Passwordless authentication
- Self-service password reset
- Auditing and reporting

## Identity Provider Role
A central identity provider:
- stores and manages identity information
- authenticates users and applications
- issues tokens
- supports authorization and auditing

## Tokens
- ID token: proves sign-in
- Access token: grants access to protected resources

## Federation
Federation allows trust between different identity providers so users can access resources across boundaries without separate credentials.

## Active Directory and Microsoft Entra
- AD DS is designed for on-prem domain-based environments
- Microsoft Entra ID supports modern authentication for cloud and hybrid identity

## Single Sign-On
SSO lets users authenticate once and access multiple applications.
