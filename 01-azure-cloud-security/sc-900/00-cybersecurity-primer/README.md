# Cybersecurity Fundamentals (SC-900 Prerequisite)

This section covers foundational cybersecurity concepts required before studying SC-900.

## Topics Covered
- Threat landscape & attack vectors
- Malware types and behaviors
- Mitigation strategies
- Cryptography fundamentals
- Authentication & Authorization
- Identity attacks
- Digital certificates & signatures

## Why This Matters
These concepts form the base of:
- Zero Trust
- Microsoft Entra ID
- Cloud Security
- SOC Operations
