# 📊 Benefits of Cloud Computing

## Scalability
- Vertical scaling: increase resources of a machine
- Horizontal scaling: add more instances

## Reliability
- Systems can recover from failures and continue working

## Predictability
- Better forecasting for performance and cost

## Security & Governance
- Built-in compliance, policies, and governance tools

## Manageability
- Resources can be managed through portal, CLI, PowerShell, and APIs
