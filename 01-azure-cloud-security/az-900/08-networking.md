# 🌐 Azure Networking

## Virtual Network (VNet)
Connect Azure resources securely.

## Subnets
Segment a VNet into smaller networks.

## Network Security Groups (NSG)
Control inbound and outbound traffic.

## Public vs Private Endpoints
- Public endpoint: reachable from the internet
- Private endpoint: reachable only inside a private network

## VPN Gateway & ExpressRoute
Connect on-premises environments to Azure.
