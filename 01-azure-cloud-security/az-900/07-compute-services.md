# 💻 Azure Compute Services

## Virtual Machines
- Full control
- Flexible for many workloads

## VM Scale Sets
- Manage and scale groups of identical VMs automatically

## Containers
- Lightweight and fast to deploy
- Useful for modern application workloads

## Azure Functions
- Serverless and event-driven
- Pay only for execution time

## App Service
- Host web apps and APIs without managing infrastructure
