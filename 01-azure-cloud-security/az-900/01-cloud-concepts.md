# ☁️ Cloud Concepts

## What is Cloud Computing?
Cloud computing is the delivery of computing services such as virtual machines, storage, databases, and networking over the internet.

## Shared Responsibility Model

### Customer is responsible for:
- Data
- Identities and access
- Devices

### Cloud provider is responsible for:
- Physical datacenter
- Physical network
- Physical hosts

## Key Insight
The level of responsibility depends on the service type: IaaS, PaaS, or SaaS.
