# 🧪 Hands-on Labs

## Lab: Create Azure VM + Web Server

### Steps
1. Create a resource group
2. Create a Linux VM
3. Install Nginx
4. Configure NSG to allow port 80

### Key Learning
- VM provisioning
- Network security basics
- Access troubleshooting

## Insight
Without allowing inbound HTTP on port 80, the web server stays unreachable from the internet.
